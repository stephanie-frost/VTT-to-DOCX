# VTT-to-DOCX
RShiny App to convert zoom VTT files to DOCX files

## How to use
To use the RShiny App, you must have a .txt file with at least ONE complete path name to a .vtt file. 

You may have multiple paths to .vtt files for batch conversion of transcripts.

## Path names
Path names must have ZERO spaces. The conversion **will not work** if the path has spaces.
